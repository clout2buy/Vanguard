# Repair atomic ledger transactions

Repair `src/ledger.mjs`. `applyTransactions(initialBalances, transactions)` must return a new plain object containing integer-cent balances after applying the complete batch. It must support `deposit`, `withdraw`, and `transfer` transactions; require every referenced account to already exist; reject non-positive or non-integer cent amounts; reject overdrafts, malformed transactions, and unknown transaction types; and never mutate either input. The operation is atomic: if any transaction is invalid, no observable input state may change. Preserve the existing export.
